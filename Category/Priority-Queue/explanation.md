## 힙 큐 (Heap Queue)

* 힙 큐는 힙(Heap) 자료구조를 기반으로 한 우선순위 큐(Priority Queue)의 일종이다.

* Python에서는 heapq 모듈을 통해 최소 힙(Min Heap)을 기본으로 제공하며, 이를 활용하여 우선순위가 높은 요소를 빠르게 찾고 처리할 수 있다.

-----

### 장점

1. 효율적인 삽입과 삭제
    * 삽입과 삭제 연산이 O(log N)의 시간 복잡도를 가짐
    * 일반적인 리스트에서 최솟값/최댓값을 찾는 O(N)보다 빠름

2. 자동 정렬 유지
    * 항상 최소(또는 최대) 값을 빠르게 접근할 수 있음

3. 우선순위 기반 처리
    * 작업 스케줄링, 다익스트라 알고리즘 등의 우선순위가 중요한 문제에서 활용 가능

-----

### 단점

1. 정렬된 상태로 유지되지만 완전한 정렬이 아님
    * 힙은 완전 이진 트리 형태를 유지하지만, 정렬된 배열과 달리 항상 오름차순/내림차순으로 정렬된 것은 아님

2. 인덱스 접근이 비효율적
    * 특정 인덱스의 값을 빠르게 수정하거나 탐색하는 것이 어려움 (O(N))

3. 기본적으로 최소 힙만 제공됨
    * 최대 힙을 사용하려면 '-value'를 넣거나 직접 구현해야 함
