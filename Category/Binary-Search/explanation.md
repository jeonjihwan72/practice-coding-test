## 이분 탐색

### 정의

* 이분 탐색은 정렬된 배열에서 특정 값을 효율적으로 찾기 위한 알고리즘이다. 배열의 중간 값을 기준으로 탐색 범위를 절반으로 줄여가면서 원하는 값을 찾는다.

-----

### 장점

1. 빠른 검색 속도 : 시간 복잡도가 O(log n)이므로, 큰 데이터에서도 빠르게 탐색할 수 있다.

2. 메모리 효율성 : 추가적인 메모리를 거의 사용하지 않는다. (재귀 구현 시에는 스택 메모리 사용 가능)

3. 정확성 : 순차 탐색보다 비교 횟수가 적어 정확한 결과를 얻을 가능성이 높다.

-----

### 단점

1. 정렬이 필요함 : 탐색을 수행하기 전에 배열이 정렬되어 있어야 한다.

2. 동적 데이터 처리 어려움 : 새로운 데이터가 추가되면 다시 정렬해야 하므로 유지보수가 어렵다.

3. 배열 기반 탐색만 가능 : 연결 리스트처럼 인덱스로 접근할 수 없는 자료구조에서는 비효율적이다.
